## Bitcoin-Price-Prediction-System

# Problem Statement:

Predicting Bitcoin's future market price using historical trading data and blockchain metrics to enable data-driven investment decisions.

# Problem Description

Bitcoin's extreme volatility makes price forecasting difficult. This project leverages historical price trends (highs, lows, closing prices) and blockchain network metrics (e.g., market cap, trade volume, mining difficulty) to analyze market dynamics. The core task involves cleaning time-series data, performing exploratory data analysis (EDA), and training a Linear Regression model to map key network features to Bitcoin's market price (btc_market_price).

# Expected Outcomes

Predictive Model: A functional Linear Regression model trained to forecast Bitcoin prices based on historical test data.
Feature Insights: Identification of the most critical variables (e.g., market cap, hash rate) driving Bitcoin price fluctuations.
Reproducible Pipeline: An end-to-end Jupyter Notebook documenting the complete data science process from data ingestion to model prediction.
